## Musl libc and x32 ABI
Start of the development of Musl on the 11th February 2011.
-------------------------------------------------------------------
The x32 ABI is part of Linux 3.4, released on the 20th May 2012.
This "Shellgen Ports" for muslx32 (x86_x32) with support other abi.
